# Premium Coffee Website ☕

A modern and responsive coffee shop website built using HTML, CSS, and JavaScript.  
This project focuses on creating an attractive user interface, smooth responsiveness, interactive navigation, and a professional coffee shop experience for users.

---

## 🚀 Project Overview

The Premium Coffee Website is a frontend project designed to practice modern web development concepts including:

- Responsive Website Design
- Flexbox Layouts
- CSS Variables
- Mobile Navigation Menu
- Swiper.js Testimonial Slider
- Smooth User Interface
- Interactive Hover Effects
- Clean UI/UX Design

The website represents a modern coffee café with sections like Hero, About, Menu, and Testimonials.

---

## ✨ Features
